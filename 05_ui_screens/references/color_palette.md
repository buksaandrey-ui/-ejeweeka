## СВОДНАЯ ТАБЛИЦА — Финальные Design Tokens
> Верифицировано визуально по 4 референсным скриншотам (2026-04-14)
> A = из скриншотов · B = из брендбука · NEW = не попало ни в одну схему

```
// Backgrounds
--color-bg-app:       #FAFAFA   /* ← B: чистый белый, не кремовый */
--color-surface:      #FFFFFF   /* белые карточки */

// Primary Orange (CTA, таб, акцент)
--color-primary:      #4C1D95   /* ← A */
--cta-gradient:       linear-gradient(135deg, #F59520, #E07018)  /* ← A */
--ring-gradient:      linear-gradient(135deg, #F59520, #E07018)  /* ← NEW: кольцо калорий = как кнопки */

// Gold (Premium)
--gold-gradient:      linear-gradient(135deg, #FFD700, #C8A96E)  /* ← B: Gold-бейдж */

// Макро-нутриенты (прогресс-бары)
--color-protein:      #52B044   /* ← A: яркий зелёный */
--color-fat:          #F09030   /* ← A: оранжевый */
--color-carbs:        #42A5F5   /* ← NEW: синий (не жёлто-зелёный!) */

// Доменные цвета
--color-sleep:        #9575CD   /* ← B: лавандовый/фиолетовый */
--color-hydration:    #42A5F5   /* ← B: голубой */
--color-workout:      #E57373   /* ← брендбук */

// Зелёный акцент (чипы, успех)
--color-green:        #52B044   /* чип «94% уверенность», бары успеха */
--color-green-light:  #E8F5E2   /* фон зелёных чипов */
--color-toggle-on:    #34C759   /* ← A: iOS-зелёный для тогла ON */

// Чипы нутриентов (экран Фото-анализ)
--chip-kcal-bg:  #FFF0E5;  --chip-kcal-fg:  #E07018   /* оранжевый */
--chip-prot-bg:  #EEF8E8;  --chip-prot-fg:  #52B044   /* зелёный */
--chip-fat-bg:   #FFF3E0;  --chip-fat-fg:   #F09030   /* янтарный */
--chip-carb-bg:  #E3F2FD;  --chip-carb-fg:  #1E88E5   /* ← NEW: сине-голубой */

// Текст
--color-text-primary: #2E2E2E
--color-text-body:    #2E2E2E
--color-text-muted:   #757575

// Borders & Dividers
--color-divider:      #E0E0E0
--color-border-input: #E0E0E0

// Tab Bar
--color-tab-active:   #4C1D95
--color-tab-inactive: #9E9E9E
```
