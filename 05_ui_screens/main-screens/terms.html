Условия использования

Дата вступления в силу: 23 апреля 2026 г.

Настоящие Условия регулируют использование мобильного приложения Health Code (далее — «Приложение», «Сервис»). Скачивая или используя Приложение, вы соглашаетесь с настоящими Условиями.

1. Описание сервиса

Health Code — мобильное приложение для персонализированного планирования питания на основе искусственного интеллекта. Сервис предоставляет:

  • Индивидуальный план питания на 7 дней
  • Рецепты с пошаговыми инструкциями
  • Расписание приёма витаминов
  • Список покупок
  • Анализ питательной ценности по фотографии еды
  • AI-чат для консультаций по питанию

2. Медицинский дисклеймер

⚠️ ВАЖНО: Health Code НЕ является медицинским устройством, НЕ ставит диагнозы и НЕ заменяет консультацию врача.

Информация в Приложении носит исключительно ознакомительный характер и собрана на основе открытых wellness-рекомендаций и нутрициологических баз данных.

Перед изменением рациона питания, приёмом витаминов или добавок обязательно проконсультируйтесь с лечащим врачом.

При наличии хронических заболеваний (диабет, заболевания почек, сердечно-сосудистые заболевания, расстройства пищевого поведения) использование Приложения допускается только с одобрения лечащего врача.

3. Статусы доступа

Base: Базовый план на 3 дня, трекер воды, дневник питания.
Black (490 ₽/мес): Полный план на 7 дней, список покупок, все рецепты.
Gold (990 ₽/мес): Всё из Black + фото-анализ, витамины, AI-отчёты, темы оформления.
  • Статусы продлеваются автоматически. Вы можете изменить статус в любой момент через Telegram-ассистента (@healthcode_bot).
  • Управление статусом осуществляется через Telegram-бота.
  • Мы не обрабатываем платёжные данные напрямую.
  • Возврат средств осуществляется через Telegram-ассистента.

4. Пробный период

Новые пользователи получают 3 дня полного доступа ко всем функциям Gold. По истечении пробного периода доступ переключается на базовый статус (Base).

5. Ограничения использования

Запрещается:

  • Использовать Приложение в коммерческих целях без письменного разрешения
  • Декомпилировать, модифицировать или реверс-инжинирить Приложение
  • Использовать Приложение для создания конкурирующего продукта
  • Передавать данные статуса третьим лицам

  • Потерю данных при удалении приложения (данные хранятся только на устройстве)
