# ⛔ ПЛАТЁЖНЫЕ ОГРАНИЧЕНИЯ — РОССИЯ, МАЙ 2026

> **Дата фиксации**: 2026-05-06
> **Статус**: ДЕЙСТВУЕТ. Учитывать ВО ВСЕХ решениях.

---

## 1. IAP (In-App Purchase) — НЕ РАБОТАЕТ В РФ

- Apple IAP полностью **неработоспособен** для российских пользователей.
- Последние способы пополнения баланса Apple ID (МТС, Билайн) **отключены**.
- Google Play Billing — аналогично ограничен.
- **Вывод**: IAP НЕ является каналом монетизации для РФ-аудитории. Любые in-app purchase flows бесполезны.

## 2. Telegram — СЕРЬЁЗНЫЕ ПРОБЛЕМЫ В РФ

- На май 2026 года Telegram испытывает **значительные проблемы с доступностью** в РФ.
- Нельзя полагаться на Telegram как на **критический канал** (оплата, восстановление, инвайты).
- Telegram может использоваться ТОЛЬКО как **опциональный/резервный** канал.

## 3. Единственный рабочий канал оплаты — ВЕБ + ЮKassa

| Канал | Статус | Роль |
|---|---|---|
| **Веб (ЮKassa)** | ✅ Работает | **ЕДИНСТВЕННЫЙ основной канал** |
| IAP (Apple/Google) | ❌ Мёртв | НЕ использовать |
| Telegram-бот | ⚠️ Нестабилен | Только резервный/опциональный |

## 4. Архитектурные следствия

- **Оплата**: Только через веб-страницу `/subscribe` → ЮKassa API
- **Восстановление статуса**: Через веб `/restore` (НЕ через Telegram)
- **Family-инвайты**: In-app генерация кода → ввод кода in-app (НЕ через Telegram)
- **Поддержка**: Email + in-app (НЕ только Telegram)
- **Deep links из приложения**: `url_launcher` → `https://сайт/subscribe?uuid=XXX`
