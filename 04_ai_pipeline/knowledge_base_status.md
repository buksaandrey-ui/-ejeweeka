## Интеграция в продуктовый пайплайн

```
Frontend (O-1..O-16) → buildProfilePayload()
        ↓
POST /generate → plan.py
        ↓
1. Формирует search_query из: goal + diseases + symptoms + womens_health
2. search_knowledge(db, query, limit=5) → pgvector cosine search
3. Инжектит в промпт: [ЭКСПЕРТНЫЕ МЕДИЦИНСКИЕ РЕКОМЕНДАЦИИ ИЗ RAG БАЗЫ]
4. Gemini генерирует план на основе RAG + profile + guardrails
5. Пост-валидация: allergen_warnings
        ↓
JSON → Frontend dashboard (H-1, P-1..P-5)
```
