> Последнее обновление: 2026-04-14
> Версия: 2.2 (solo-founder + AI)

---

## 1. Что это за документ

План разработки MVP (минимально жизнеспособного продукта) AIDiet. Ключевая особенность: проект разрабатывается ОДНИМ человеком с помощью оркестрации нейросетей и AI-инструментов разработки. Без наёмных разработчиков и дизайнеров.

---

## 2. Что входит в MVP

### Включено в MVP (делаем сейчас)

- Онбординг (17 экранов, O-1 — O-17): сбор данных пользователя
- Интеграция Gemini API: RAG-поиск по базе знаний врачей → компиляция персонального плана
- Недельный план с 4 приёмами пищи + рецепты
- Витаминное расписание с проверкой совместимости
- Дневник (вес, вода, настроение, сон, витамины)
- Список покупок (автогенерация из плана)
- Photo Analysis v1.1 (распознавание блюд по фото) — Gold
- Еженедельный Умный отчёт — Black/Gold
- Health Connect (сон, шаги, тренировки, вес)
- Подписки (Free / Black / Gold / Group Gold) через ЮKassa
- Push-уведомления (еда, витамины, вода, тренировки)

### Отложено на v1.1 (после MVP)

- Детальные тренировочные программы (с видео)
- Геймификация (достижения, награды)
- Социальные функции (поделиться планом)
- Интеграция с магазинами (автозаказ продуктов)
- Web-версия личного кабинета

## 3. Модель разработки: Solo Founder + AI

### Кто делает

Один человек — Андрей (продукт, маркетинг, контент, разработка, тестирование, запуск).

### Как это возможно

Вместо команды из 3-5 разработчиков используется оркестрация AI-инструментов:

**Google Antigravity (IDE)** — основная среда разработки. AI-агенты на базе Gemini 3 Pro автономно пишут код, тестируют через встроенный браузер, деплоят. Бесплатно.

**Claude (Anthropic)** — архитектура, документация, генерация всех UI-экранов (через UI UX Pro Max + Frontend Design skill), системные промпты для Recommendation Engine, ревью кода, решение нестандартных задач.

**ChatGPT (OpenAI)** — альтернативная проверка кода, генерация контента, помощь с маркетинговыми текстами.

**Gemini (Google)** — встроен в Antigravity, основной агент для написания кода. Также используется для работы с базой знаний, генерации планов питания (RAG + компиляция), Photo Analysis v1.1 (Gemini Vision), еженедельных Умный отчётов.

**Grok (xAI)** — дополнительная верификация решений, альтернативный взгляд на архитектурные вопросы.

### Принцип работы

1. Андрей формулирует задачу (промпт)
2. AI-инструмент генерирует решение (код, дизайн, текст)
3. Андрей проверяет, корректирует, утверждает
4. Переходит к следующей задаче

По сути: Андрей — «продукт-менеджер и архитектор», AI — «команда исполнителей».

### Фаза 2 — Backend (серверная часть)
Длительность: 3-4 недели
Инструменты: Antigravity + Claude + Gemini

Неделя 1: Фундамент
- Развернуть PostgreSQL, Redis в Yandex Cloud
- Создать таблицы (скопировать schema.sql)
- Auth Service: анонимный вход, OTP, OAuth (VK, Яндекс)
- JWT-авторизация (access + refresh токены)
- Базовое API: профиль, CRUD

Неделя 2: Основная логика
- Core API: план питания, дневник, список покупок
- Интеграция с LLM API: генерация плана (RAG по базе знаний врачей)
- Заполнение справочника блюд и ингредиентов (200-300 блюд)
- Проверка совместимости витаминов/лекарств

Неделя 3: Расширенные функции
- Photo Analysis v1.1 (Gemini Vision + Object Storage)
- Умный отчёт (еженедельная генерация)
- Payment Service (ЮKassa: подписки, вебхуки)
- Push Service (FCM + APNs)

Неделя 4: Тестирование и отладка backend
- Тестирование всех эндпоинтов
- Нагрузочное тестирование
- Исправление багов

Вход: architecture.md + schema.sql + openapi.yaml + ai-pipeline.md
Выход: работающий API, доступный по URL

### Инфраструктура (ежемесячно после запуска)
Yandex Cloud (PostgreSQL + Redis + K8s): 5 000 - 15 000 руб/мес
Yandex Object Storage (фото): 500 - 2 000 руб/мес

LLM API (Claude/GPT для генерации планов): 3 000 - 15 000 руб/мес
Зависит от количества пользователей. При 100 активных — ~3 000, при 1000 — ~15 000.

### Разовые расходы
Apple Developer Program: $99/год (~9 000 руб)
Google Play Developer: $25 разово (~2 300 руб)
ЮKassa подключение: бесплатно (комиссия 3.5% с каждого платежа)

### Итого

- AI-подписки: 3 600 - 7 200 руб (если нужны платные тарифы Claude/ChatGPT)

## 8. Риски и как их снижаем

Риск: LLM генерирует некорректные рекомендации
Снижение: RAG-поиск по базе знаний 20+ врачей + дисклеймер «не заменяет врача»

Риск: Высокая стоимость AI API при росте пользователей
Снижение: кеширование, более дешёвые модели для рутинных проверок, собственные модели в будущем

Риск: Долгая модерация App Store
Снижение: заранее подготовить все документы, политики, описания

Риск: Низкая конверсия Free → Black/Gold
Снижение: мягкий statuswall, trial-период, ценность видна уже в White-версии

Риск: Один человек = узкое место (болезнь, выгорание)
Снижение: автоматизировать максимум процессов, документировать всё в GitHub (чтобы в будущем можно было подключить людей), работать итерациями (не пытаться сделать всё идеально сразу)

Риск: Antigravity выйдет из бесплатного preview
Снижение: код, созданный в Antigravity — обычный код (Flutter/React Native), его можно продолжить разрабатывать в любой IDE (VS Code, Cursor, Windsurf)
