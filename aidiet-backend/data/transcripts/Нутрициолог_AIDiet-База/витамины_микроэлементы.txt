Витамины и микронутриенты — рекомендации на основе анализов крови

Витамин D3. Оптимальный уровень 25(OH)D в крови: 40–60 нг/мл. При уровне ниже 30 нг/мл — дефицит, требующий коррекции. Дозировка: поддерживающая 1000–2000 МЕ/день, при дефиците 5000–10000 МЕ/день курсом 8–12 недель. Принимать с жирной пищей (жирорастворимый). Не сочетать с кальцием одновременно.

Железо. Ферритин — главный маркер запасов железа. Оптимальный уровень: 40–100 нг/мл для женщин, 60–150 нг/мл для мужчин. При ферритине ниже 30 нг/мл — латентный дефицит. Симптомы: усталость, выпадение волос, бледность, одышка. Принимать натощак с витамином С для лучшего усвоения. Не сочетать с кальцием, кофе, чаем (снижают усвоение).

Магний. Суточная потребность: 400–420 мг для мужчин, 310–320 мг для женщин. Дефицит распространён (до 60% населения). Симптомы: мышечные судороги, бессонница, тревожность, головные боли. Лучшие формы: магний бисглицинат (для сна и нервной системы), магний таурат (для сердца), магний цитрат (для ЖКТ). Принимать вечером. Не сочетать с кальцием.

Омега-3 (EPA + DHA). Суточная потребность: минимум 1000 мг EPA+DHA. Для противовоспалительного эффекта: 2000–3000 мг. Источники: жирная рыба (лосось, сардины, скумбрия), рыбий жир. Принимать с едой. Важно соотношение EPA:DHA — при воспалении лучше EPA-доминанта, для мозга — DHA-доминанта.

Витамин B12. Оптимальный уровень в крови: 500–900 пг/мл. Критически важен для веганов и вегетарианцев (не содержится в растительной пище). Форма метилкобаламин предпочтительнее цианокобаламина.

Цинк. Суточная потребность: 11 мг для мужчин, 8 мг для женщин. Важен для иммунитета, заживления ран, вкусовых рецепторов. Не принимать одновременно с железом и кальцием.

Йод. Суточная потребность: 150 мкг. Критически важен для щитовидной железы. В эндемичных зонах (большая часть России) — рекомендована йодированная соль или добавки. Противопоказан при тиреотоксикозе.
